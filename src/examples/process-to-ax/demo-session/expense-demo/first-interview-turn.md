# 독립 전방 테스트: 첫 인터뷰 응답

입력 파일: `src/examples/work-descriptions/expense-reimbursement-raw.txt`

```text
[진행] 범위 초안 · 기본 흐름 0/? · 분기 0/? · 전체 확인 ○ · AX 분석 잠금

이번에 반영한 내용
- 업무명: 출장비 정산 업무
- 시작: 직원이 출장을 다녀온 뒤 영수증을 모으기 시작함
- 원문에 적힌 처리 구간은 범위 초안으로 저장했으며, 종료 조건은 미확인으로 남김

다음 질문
이 출장비 정산 업무가 끝났다고 보는 시점은 언제인가요?

명령: 확정 · 수정 · 이전 · 요약 · 완료 · 재개
```

확인 결과: 첫 응답에서 질문은 하나였고, 종료 조건을 추측하지 않았으며, 전체 확정 전 AX 분석 잠금을 유지했습니다.
