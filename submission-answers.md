# 질문 5문항 답변 초안

## 1. 무엇을, 누가, 어떤 상황에서 쓰나요?

삼일PwC 페르소나의 상장사 서비스 기회 추천을 보조하는 Codex 플러그인입니다. 회계법인 감사·영업·리스크 담당자가 상장사를 검색해 최근 완료연도 확인 감사인, 최근 감사인 이력, 동일 감사인 연속 이력, 최신 데이터 공백, 현재 선임기간 원문 확인 필요성, 감사용역 보수·시간, 임원 주요경력을 확인하고, 사람인 채용공고 키워드로 Assurance/Risk, Tax, Deals, Capital Markets, Finance Transformation 수요 신호를 조기 탐지할 때 사용합니다. 실제 서비스 맥락에서는 회계법인 측 인력의 업종 감사경험·도메인 지식, 독립성 제한, 기존 관계, 감사위원회·감사·사외이사 후보군과의 합법적 접점 신호를 함께 고려할 수 있습니다.

## 2. 왜 이 문제를 선택했나요?

회계법인의 본질적 업무는 외부감사이지만, 실제 성장 기회는 감사, Tax, Deals, 내부통제, 재무시스템 자문 중 어느 경로가 더 적합한지 먼저 판단하는 데 있습니다. 공시는 확정 이벤트 확인에는 유용하지만 영업 타이밍으로는 늦을 수 있으므로, 채용공고의 내부회계·연결결산·이전가격·M&A·IPO·ERP 키워드를 선제 신호로 사용합니다. 이후 OpenDART로 최근 완료연도 감사인, 감사보수, 임원·감사위원회 정보를 확인하고, 현재 선임 결과는 회사 원문으로 검증하도록 설계했습니다.

## 3. 플러그인은 어떻게 작동하나요?

플러그인은 `audit-radar` 스킬과 `scripts/audit_radar.py` 실행 스크립트로 구성됩니다. OpenDART 고유번호 목록에서 종목코드가 있는 상장사 후보만 검색하고, `회계감사인의 명칭 및 감사의견` API를 여러 사업연도에 걸쳐 조회합니다. `감사용역체결현황` API로 감사보수·감사시간을, `임원 현황` API로 대표이사·사외이사·감사위원회 관련 주요경력을 함께 표시합니다. 2월·3월 등 비12월 결산회사의 최신 완료연도도 포함하고, 사업보고서가 있는데 감사인 구조화 데이터가 비어 있으면 상단에 데이터 공백을 경고합니다. 동일 감사인 이력만으로 계약 시작·종료나 주기적 지정 D-day를 산출하지 않으며, 공식 선임공고가 확인되지 않으면 날짜 대신 원문 확인 상태를 표시합니다. 추가로 `jobs` 명령은 사람인 Job Search API를 seed 키워드별로 호출해 관련 채용공고를 수집하고, 키워드 룰로 Assurance/Risk, Tax, Deals, Capital Markets, Finance Transformation 신호를 점수화합니다. 이후 삼일PwC firm context와 독립성 검토 필요성을 함께 표시합니다.

## 4. AI를 어떻게 활용했나요?

AI를 사용해 과제 요구사항을 해석하고, 회계법인의 감사영업 맥락, OpenDART 공개 API, 외부감사인 주기적 지정 제도에 대한 공개 자료를 조사했습니다. 이후 문제 범위를 상장사로 좁혀 감사인 매칭·연속 이력·감사보수·임원 주요경력·원문 확인 상태 기능으로 설계하고, 단순 조회 서비스가 아니라 삼일PwC 페르소나의 감사영업 추천 시스템으로 제품 방향을 재정의했습니다. Codex 플러그인 manifest, 스킬 문서, Python 기반 OpenDART 연동 코드, 웹 UI, README와 답변 초안을 AI와 협업해 작성했습니다.

## 5. 어떻게 검증했나요?

API 키 없이도 동작 확인이 가능한 합성 감사인 이력, 임원 주요경력 샘플, 삼일PwC firm context로 데이터 표시와 추천등급 산정 로직을 검증합니다. 실제 데이터는 삼성전자뿐 아니라 SK하이닉스, 현대자동차, KB금융, NAVER, POSCO홀딩스, JTC 등 12개사를 표본검사하고 회사 IR·KRX·DART 원문과 대조했습니다. 이 과정에서 동일 감사인의 미래 3년 자동 연장, 현대자동차 최신 감사인 데이터 공백, JTC 비12월 결산 최신연도 누락, 정정공시 중복, 공시 제출인명 오용을 발견해 보수적으로 수정했습니다. 공개 데모에는 실명 개인정보나 삼일 내부 데이터가 없습니다. Render 공개 배포 URL(`https://samil-audit-radar.onrender.com`)에서 API 응답과 웹 UI 렌더링을 확인하고, 자동 회귀검사와 plugin validator로 코드 및 플러그인 구조를 함께 검증합니다.
